# Зарплатный счётчик

Виджеты, показывающие как «капает» зарплата в реальном времени.

## Запуск

```bash
cd zp_demo
pip install -r requirements.txt   # или: python3 -m pip install -r requirements.txt
python app.py                      # или: python3 app.py
```

Открой в браузере: **http://127.0.0.1:5001**

## Как работает

- Указываешь зарплату в месяц (после налогов)
- График: 9:00–17:00, обед 13:00–14:00 (счётчик не тикает)
- Только рабочие дни по календарю РФ (выходные и праздники не считаются)
- Можно добавить несколько работ — сверху показывается суммарный счётчик
