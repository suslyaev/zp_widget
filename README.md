# zp-widget

`zp-widget` — веб-приложение с живыми виджетами заработка.
Показывает, сколько уже заработано за день/неделю/месяц/год, умеет учитывать график, налоги, отпуска, допвыплаты, валюты и замеры времени.

## Что умеет

- Несколько работ одновременно.
- Разные модели ставки: в месяц, в день, в час.
- Гибкий налог в процентах (float) для работы и допвыплат.
- Допвыплаты с периодом, налогом и графиком (наследуемым или отдельным).
- Отпуска и их влияние на расчёт.
- Календарь для `5x2`:
  - праздничные дни по стране из открытого API,
  - fallback на `пн-пт`, если API недоступен,
  - ручные переопределения рабочих/выходных дней.
- Валюты у работы и у виджета + конвертация по курсам.
- Ручные курсы как fallback при проблемах с интернетом/API.
- Секундомеры для активности и история замеров.
- Шаринг виджета и замеров (картинка + ссылка/текст).

## Важно

Цифры в виджете показывают оценку фактически заработанного за выбранный период.
Это не гарант суммы «к зачислению сегодня на карту» (выплаты часто идут аванс/расчёт по внутреннему графику компании).

## Локальный запуск

```bash
python3 -m pip install -r requirements.txt
python3 app.py
```

По умолчанию приложение доступно на:

- `http://127.0.0.1:5001` (локально на компьютере)

Если нужно открыть с телефона в той же Wi-Fi сети, запускайте на `0.0.0.0`
и открывайте `http://<ваш-lan-ip>:5001`.

## Стек

- Backend: `Flask` (Python)
- Frontend: vanilla HTML/CSS/JS
- Хранение пользовательских данных: `localStorage` в браузере
